Guías de Solución

Internet Lento

Pasos para mejorar la velocidad de tu conexión

  1. Reinicia tu router desconectándolo por 30 segundos
  2. Verifica que no hay descargas activas
  3. Conecta tu dispositivo por cable Ethernet
  4. Cierra aplicaciones que consuman ancho de banda
  5. Realiza un test de velocidad en wifiman.com

Sin Conexión a Internet

Soluciona problemas de conectividad

  1. Verifica que todos los cables estén conectados
  2. Revisa las luces del router (deben estar verdes)
  3. Reinicia el router por 2 minutos
  4. Verifica la configuración de red en tu dispositivo
  5. Contacta soporte si el problema persiste

Problemas de WiFi

Mejora tu conexión inalámbrica

  1. Acércate al router para mejor señal
  2. Reinicia la conexión WiFi en tu dispositivo
  3. Olvida y vuelve a conectar la red WiFi
  4. Cambia el canal WiFi en la configuración del router
  5. Considera usar la banda de 5GHz si está disponible

Cómo Reiniciar Router

Proceso paso a paso para reiniciar tu equipo

  1. Desconecta el cable de alimentación del router
  2. Espera 30 segundos
  3. Vuelve a conectar el cable de alimentación
  4. Espera 2-3 minutos hasta que las luces se estabilicen
  5. Prueba tu conexión

Mejorar Cobertura WiFi

Optimiza la señal en toda tu casa

  1. Coloca el router en posición central y elevada
  2. Aleja el router de dispositivos que interfieran
  3. Actualiza el firmware del router
  4. Considera un repetidor WiFi para áreas lejanas
  5. Usa aplicaciones como WiFiMan para medir señal

Test de Velocidad

Cómo realizar un test de velocidad preciso

1

Conecta por cable Ethernet

Para obtener resultados más precisos, conecta tu dispositivo directamente al router con un cable Ethernet.

2

Cierra otras aplicaciones

Asegúrate de cerrar aplicaciones que usen internet como streaming, descargas o videollamadas.

3

Ejecuta el test

Haz clic en el botón de abajo para ir a wifiman.com y realizar el test de velocidad.

4

Toma captura y envía

Toma una captura de pantalla del resultado y envíala a nuestro soporte técnico.

Ir a WiFiMan (Recomendado) Alternativa: Fast.com

Qué esperar en tu test:

Excelente: 80-100% de la velocidad contratada
Bueno: 60-79% de la velocidad contratada
Revisar: Menos del 60% - Contacta soporte

Base de Conocimientos

Para reiniciar tu router correctamente:

  1. Desconecta el cable de alimentación del router
  2. Espera 30 segundos
  3. Vuelve a conectar el cable
  4. Espera 2-3 minutos hasta que las luces se estabilicen

Las causas más comunes de internet lento incluyen:

  • Muchos dispositivos conectados simultáneamente
  • Aplicaciones consumiendo ancho de banda en segundo plano
  • Interferencia en la señal WiFi
  • Router que necesita reiniciarse
  • Problemas en la línea o equipos

Para mejorar tu señal WiFi:

  • Posiciona el router en un lugar central y elevado
  • Aleja el router de microondas y otros dispositivos
  • Usa la banda de 5GHz si está disponible
  • Considera un repetidor para áreas grandes
  • Actualiza el firmware del router regularmente

Para conectar nuevos dispositivos a tu WiFi:

  1. En tu dispositivo, ve a Configuración > WiFi
  2. Selecciona tu red WiFi (nombre del router)
  3. Ingresa la contraseña WiFi (ubicada en la etiqueta del router)
  4. Confirma la conexión
  5. Si tienes problemas, reinicia el router y intenta nuevamente
